# LeaFrice — Website Statis (GitHub Pages)

## Cara pakai
1. Buat repository baru di GitHub (mis. `leafrice-website`).
2. Clone atau upload isi folder ini ke repository.
3. Pastikan file gambar ada di `assets/`:
   - `assets/logo.jpg`
   - `assets/background.jpg`
   - `assets/Nasi Ayam Jeruk LeaFrice.jpg`
4. Commit & push ke branch `main`.
5. Aktifkan GitHub Pages: Settings > Pages > Source: `main` branch > root (`/`) > Save.
6. Tunggu beberapa menit — situs tersedia di `https://<username>.github.io/<repo>`.

## Ganti detail
- Nama pemilik, nomor WA, email, detail rekening, dan embed YouTube/Maps: edit `index.html`.
- Tambah produk: duplikat blok `.product-card` dan sesuaikan gambar & harga.
- Untuk QRIS: taruh gambar QRIS di `assets/` dan tampilkan di bagian Pembayaran.
